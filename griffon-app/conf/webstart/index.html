Instalasi Melalui Java Web Start

Invento adalah demo aplikasi yang dibuat dengan menggunakan Groovy, Griffon dan simple-jpa untuk mengelola inventory dan transaksi finansial Anda.

Aplikasi ini bersifat open-source dimana kode programnya dapat dibaca di GitHub simple-jpa-demo-inventory.

Karena ini adalah proyek open-source, Anda akan memperoleh garansi seumur hidup. Kode program akan terus kami perbaharui mengikuti perkembangan zaman dan kebutuhan.

Kami menghargai privasi Anda, oleh sebab itu kami tidak menyimpan data sensitif seperti isi database Anda.
